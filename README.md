# Описание
Прогрмма предназначена для использования телеграм бота как словарь с английскими словами и переводом, с возможностью удаления и добавления новых слов.

Перед использованием необходимо:
- загрузить все библиотеки из файла "requirements.txt";
- в файл "settings.ini" вставить значения:
1) токен телеграмм бота в поле - token_bot;
2) наименование БД к которой будет подключаться Python - telegram_bot;
3) пароль для доступа к БД - password;
4) при необходимости изменить поля login и localhost.

Для понимания, как использовать бота, после его создания и настройки, в телеграмме напишите команду: /help
